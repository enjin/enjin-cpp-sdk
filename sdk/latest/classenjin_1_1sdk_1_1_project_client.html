Authenticates the client using the given access token model.

Parameters
access_tokenThe access token.
Exceptions
std::runtime_errorIf access_token exclusively returns an empty optional from either its token or expiration getter.
Remarks
If this client has automatic reauthentication enabled, then this method may halt the reauthentication timer when access_token is null. Otherwise the timer will be restarted.
